Ch. 1
🧭 Vectors

A vector is an ordered list of numbers representing a point or direction in space. In 2D: [x, y]. Think of it as an arrow from the origin.

Key Concepts
  • Vectors have magnitude (length) and direction
  • Addition: tip-to-tail — move along first then second vector
  • Scalar multiplication: stretch, shrink, or flip the vector
  • Each coordinate is how far you move along each axis
v = [x, y]
|v| = √(x² + y²) ← magnitude
v + w = [x₁+x₂, y₁+y₂]
c · v = [cx, cy]
Ch. 2
🌐 Span, Basis & Linear Combos

A linear combination scales and adds vectors together. The span is all reachable points. Basis vectors are the fundamental building blocks of a space.

Key Concepts
  • Linear combination: a·v + b·w for any scalars a, b
  • Span: the set of all possible linear combinations
  • Standard basis: î = [1,0], ĵ = [0,1]
  • Linearly dependent: one vector lies in span of others (redundant)
  • Basis: minimal set of linearly independent spanning vectors
Linear combo: a·î + b·ĵ
Span(v,w) = {av + bw | a,b ∈ ℝ}
Basis of ℝ² = {î, ĵ} = {[1,0],[0,1]}
Ch. 3
🔄 Linear Transformations & Matrices

A linear transformation is a function that moves vectors while keeping grid lines parallel and evenly spaced, and the origin fixed. Every such transformation is fully described by a matrix.

Key Concepts
  • A matrix encodes where î and ĵ land after transformation
  • Apply transformation: multiply matrix by vector
  • Must satisfy: T(v+w)=T(v)+T(w) and T(cv)=cT(v)
M = [[a, b], [c, d]]
M · v = [ax+by, cx+dy]
Columns = where î, ĵ land

Ch. 4
🔗 Matrix Multiplication as Composition

Multiplying two matrices is the same as applying one transformation then another. Order matters — read right-to-left: AB means "first B, then A."

Key Concepts
  • AB ≠ BA in general (non-commutative)
  • Each column of AB = A applied to that column of B
  • Associative: (AB)C = A(BC) ✓
  • Each result entry = dot product of a row with a column
(AB)ᵢⱼ = Σₖ Aᵢₖ · Bₖⱼ
2×2 example:
[[a,b],[c,d]]·[[e,f],[g,h]]
= [[ae+bg, af+bh],[ce+dg, cf+dh]]
Ch. 5
🧊 3D Linear Transformations

The same ideas extend to 3D. A 3×3 matrix tracks where the three basis vectors î, ĵ, k̂ land, encoding any rotation, scale, or shear in 3D space.

Key Concepts
  • 3D basis: î=[1,0,0], ĵ=[0,1,0], k̂=[0,0,1]
  • 3×3 matrix = three column vectors showing new positions
  • Right-hand rule determines rotation direction
  • Composition still works: multiply 3×3 matrices
M·v where M is 3×3, v is [x,y,z]
Rotation X: [[1,0,0],[0,cosθ,-sinθ],[0,sinθ,cosθ]]
Ch. 6
📐 The Determinant

The determinant measures how much a transformation scales area (2D) or volume (3D). A negative det means orientation is flipped. Zero det means the space is squished flat.

Key Concepts
  • det = 0 → transformation squishes to lower dimension (singular!)
  • |det| = scale factor of area/volume
  • Negative det = orientation flipped (mirror image)
  • det(AB) = det(A) · det(B)
2×2: det([[a,b],[c,d]]) = ad - bc
3×3: expansion along first row
det(I) = 1 | det(flip) = -1
Ch. 7
🔍 Inverse Matrices, Column Space & Null Space
Inverse Matrix A⁻¹

Undoes a transformation. A⁻¹A = I. Exists only when det ≠ 0. Solves Ax = b as x = A⁻¹b.

A·A⁻¹ = I
2×2: A⁻¹ = 1/det · [[d,-b],[-c,a]]
Column Space

All possible outputs of Ax — the "span" of the columns. The solution to Ax=b exists only if b is in the column space.

Col(A) = {Ax | x ∈ ℝⁿ}
rank = dim of column space
Full rank → invertible
Null Space (Kernel)

All vectors that get squished to zero by A. Captures "lost" information in the transformation. Non-trivial null space → not invertible.

Null(A) = {x | Ax = 0}
rank + nullity = n
(rank-nullity theorem)
Ch. 8
↔️ Nonsquare Matrices

A nonsquare matrix maps between spaces of different dimensions. A 3×2 matrix maps 2D → 3D; a 2×3 maps 3D → 2D (like a projection).

Key Concepts
  • m×n matrix: maps n-dim input → m-dim output
  • Columns still = where each basis vector lands
  • Cannot be invertible (dimensions don't match)
  • Useful for projections, encodings, embeddings
3×2 matrix: ℝ² → ℝ³
2×3 matrix: ℝ³ → ℝ²
columns = # inputs | rows = # outputs
Ch. 9
· Dot Products & Duality

The dot product measures how much two vectors point in the same direction. It equals the product of their magnitudes and the cosine of the angle between them.

Key Concepts
  • Geometric: v·w = |v||w|cosθ
  • Algebraic: sum of element-wise products
  • v·w > 0 → same direction; < 0 → opposite; = 0 → perpendicular
  • Duality: dot product ↔ 1×n matrix (linear functional)
v·w = x₁x₂ + y₁y₂ (+ z₁z₂ in 3D)
= |v||w|cosθ
Projection of w onto v: (v·w)/|v|
Ch. 14
λ Eigenvectors & Eigenvalues

An eigenvector is a special vector that only gets scaled (not rotated) by a transformation. The eigenvalue λ is how much it gets scaled. These reveal the "essence" of a matrix — the axes around which it truly acts.

Definition
  • Av = λv — vector stays on its span
  • λ can be negative (flip), 0 (squish to origin), or complex
  • Each eigenvalue may have multiple eigenvectors (eigenspace)
How to Find Them
det(A - λI) = 0 ← characteristic eq.
Solve for λ (eigenvalues)
Then: (A - λI)v = 0 → eigenvectors
Why They Matter
  • Diagonalization: A = PDP⁻¹ (P = eigenvectors)
  • Stable axes in 3D rotations
  • PCA, PageRank, quantum states, vibrations
  • Powers of A: Aⁿ = PDⁿP⁻¹ (easy!)
