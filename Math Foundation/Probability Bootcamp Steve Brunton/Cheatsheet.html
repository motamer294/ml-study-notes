§1 · Foundations of Probability
Topic 120:05
🪙 Intro to Probability: Coins & Dice

Probability measures the likelihood of an event. Built on counting outcomes — how many ways can something happen vs. total possible outcomes.

Key Concepts
  • Sample space Ω: all possible outcomes
  • Event: a subset of the sample space
  • Coin flip: P(H) = 1/2 · Dice: P(6) = 1/6
  • Complement: P(Aᶜ) = 1 − P(A)
P(A) = favorable / total
0 ≤ P(A) ≤ 1 | P(Ω) = 1
P(A∪B) = P(A)+P(B)−P(A∩B)
Topic 213:11
🔗 Conditional Probability

The probability of A given that B has already occurred. Updates our belief based on new information — the foundation of Bayesian thinking.

Key Concepts
  • Restricts sample space to outcomes where B holds
  • Independent events: P(A|B) = P(A)
  • Multiplication rule: P(A∩B) = P(A|B)·P(B)
P(A|B) = P(A∩B) / P(B)
Independent: P(A∩B) = P(A)·P(B)
Topic 310:13
⚖️ Law of Total Probability

If events B₁…Bₙ partition the sample space, the total probability of A is the weighted sum of conditional probabilities across all partitions.

Key Concepts
  • Partition: mutually exclusive + exhaustive events
  • Lets you compute P(A) by breaking into cases
  • Essential building block for Bayes' theorem
P(A) = Σᵢ P(A|Bᵢ)·P(Bᵢ)
where B₁…Bₙ partition Ω
§2 · Bayes' Theorem
Topic 417:43
🔁 Bayes' Theorem — Inverting Conditional Probability

Bayes' Theorem lets you reverse conditional probability: if you know P(B|A), you can compute P(A|B). It updates a prior belief with new evidence to get a posterior belief.

Components
  • Prior P(A): belief before seeing evidence
  • Likelihood P(B|A): how probable is evidence given A
  • Posterior P(A|B): updated belief after evidence
  • Evidence P(B): total probability of B (normalizer)
P(A|B) = P(B|A)·P(A) / P(B)

Expanded denominator:
P(B) = P(B|A)·P(A) + P(B|Aᶜ)·P(Aᶜ)

Posterior ∝ Likelihood × Prior
Intuition
  • Start with a prior → observe data → update to posterior
  • More evidence → posterior converges to truth
Topic 512:32
🌿 Bayes' Theorem Example: Drug Testing

Classic example: a drug test is 99% accurate. Only 0.5% of people actually use drugs. If you test positive, what's the real probability you use drugs?

P(user) = 0.005 (prior)
P(+|user) = 0.99 (sensitivity)
P(+|non-user) = 0.01 (false +)

P(user|+) ≈ 33% ← surprising!
Key Lesson
  • Low base rate → high false positive rate even with accurate test
  • Prior matters enormously for interpreting test results
  • This is the base rate fallacy in action
§3 · Random Variables & Distributions
Topic 621:56
🎲 Random Variables & Distributions

A random variable assigns a number to each outcome. Distributions describe how probability is spread across values.

  • Discrete: countable values (PMF)
  • Continuous: any real value (PDF)
  • CDF: P(X ≤ x)
  • Binomial: n trials, p success
P(X=k) = C(n,k)pᵏ(1-p)ⁿ⁻ᵏ
PDF: f(x) ≥ 0, ∫f(x)dx = 1
Topic 717:15
🔔 Normal Distribution

The bell curve — arises as the limit of the Binomial for large n. Defined entirely by mean μ and std σ.

  • Symmetric, bell-shaped curve
  • 68-95-99.7 rule (σ intervals)
  • Standard normal: μ=0, σ=1
  • Z-score: z = (x−μ)/σ
f(x) = (1/σ√2π)e^(-(x-μ)²/2σ²)
X ~ N(μ, σ²)
Topic 814:34
📦 Joint Probability Distributions

Describes probability over two or more random variables simultaneously. Captures how variables relate to each other.

  • Joint PMF/PDF: P(X=x, Y=y)
  • Continuous: f(x,y) joint density
  • Independent: f(x,y) = f(x)·f(y)
  • Volume under surface = 1
P(X,Y): full joint table
ΣΣ P(X=x,Y=y) = 1
Topic 99:36
📊 Marginal & Conditional Densities

From a joint distribution, recover single-variable distributions by summing/integrating out the other variable.

  • Marginal: integrate out other variable
  • Conditional: f(x|y) = f(x,y)/f(y)
  • Marginalizing = projecting onto one axis
f_X(x) = ∫ f(x,y) dy
f(x|y) = f(x,y) / f_Y(y)
§4 · Expectation & Variance
Topic 1015:24
📍 Expected Value (Mean)

The long-run average — the "center of mass" of a distribution. Weighted average of all values by their probabilities.

  • Discrete: Σ xᵢ · P(X=xᵢ)
  • Continuous: ∫ x · f(x) dx
  • Not necessarily achievable
E[X] = Σ xᵢP(xᵢ) [discrete]
E[X] = ∫ x·f(x)dx [continuous]
Topic 1110:18
📏 Properties of Expected Value

Expectation is linear — one of the most powerful and useful properties in all of probability theory.

  • Linearity: E[aX+b] = aE[X]+b
  • E[X+Y] = E[X]+E[Y] always
  • E[XY] = E[X]E[Y] only if independent
  • E[g(X)] = ∫ g(x)f(x)dx
E[aX+bY] = aE[X] + bE[Y]
E[c] = c (constant)
Topic 1212:59
📉 Variance & Standard Deviation

Variance measures how spread out a distribution is — the expected squared deviation from the mean. Std dev is its square root (same units as X).

  • Var(X) = E[X²] − (E[X])² (shortcut!)
  • Var(aX+b) = a²Var(X)
  • σ = √Var(X)
Var(X) = E[(X−μ)²]
= E[X²] − μ²
σ(X) = √Var(X)
Topic 1311:58
📐 Exponential Distribution Example

Works through computing E[X] and Var(X) for the exponential distribution — models time between events (e.g. radioactive decay, wait times).

  • Memoryless property: P(X>s+t|X>s) = P(X>t)
  • Rate parameter λ controls shape
f(x) = λe^(−λx), x ≥ 0
E[X] = 1/λ | Var(X) = 1/λ²
Topic 1415:26
🌡️ Functions of Random Variables: Temp C↔F & Kinetic Theory

If Y = g(X), find E[Y] and Var(Y). Two concrete examples: temperature conversion (linear) and kinetic energy (quadratic).

  • Linear: Y = aX+b → E[Y]=aμ+b, Var=a²σ²
  • Nonlinear: use LOTUS (Law of the Unconscious Statistician)
  • E[g(X)] = ∫ g(x)f(x)dx
Temp: F = 9/5·C + 32
E[F] = 9/5·E[C] + 32

Kinetic: KE = ½mv²
E[KE] = ½m·E[v²] = ½m(σ²+μ²)
§5 · Central Limit Theorem & Covariance
Topic 1510:57
🏛️ Central Limit Theorem

One of the most profound results in statistics: the sum (or mean) of many independent random variables approaches a Normal distribution, regardless of the original distribution.

Key Concepts
  • Works for any distribution with finite μ and σ
  • Rule of thumb: n ≥ 30 is often "large enough"
  • Explains why the normal distribution is everywhere
  • Foundation of confidence intervals & hypothesis testing
X̄ ~ N(μ, σ²/n) as n → ∞
Standardized: Z = (X̄−μ)/(σ/√n)
Topic 1619:34
🔀 Covariance & Correlation

Covariance measures how two variables change together. Correlation normalizes this to a −1 to +1 scale for easy interpretation.

Key Concepts
  • Cov > 0: move together · Cov < 0: move opposite
  • Cov(X,X) = Var(X)
  • Correlation r: scale-free, −1 ≤ r ≤ 1
  • r = ±1 → perfect linear relationship
  • r = 0 → uncorrelated (not necessarily independent!)
Cov(X,Y) = E[XY] − E[X]E[Y]
ρ(X,Y) = Cov(X,Y)/(σ_X · σ_Y)
Topic 17—
📈 Covariance Example: Gaussian Distributions

Worked example computing covariance and correlation for jointly Gaussian variables — the most important multivariate distribution in all of statistics.

Key Concepts
  • Bivariate Normal defined by (μ_X, μ_Y, σ_X, σ_Y, ρ)
  • For Gaussians: uncorrelated ↔ independent (special!)
  • Covariance matrix Σ encodes all relationships
  • Contour plots are ellipses tilted by ρ
Σ = [[σ_X², ρσ_Xσ_Y],
    [ρσ_Xσ_Y, σ_Y² ]]
ρ=0 → circular contours
