§1 · Measures of Central Tendency
AverageMost Common
➕ Mean — Arithmetic Average

The sum of all values divided by the number of values. The "balancing point" of the data. Affected by outliers.

Example · Dataset: 2, 4, 4, 4, 5, 5, 7, 9

Sum = 2+4+4+4+5+5+7+9 = 40
n = 8
μ = Σxᵢ / n = 40 / 8 = 5.0
Formula
Population: μ = Σxᵢ / N
Sample: x̄ = Σxᵢ / n
Watch out
  • Sensitive to outliers — one extreme value shifts it
  • Best for symmetric distributions
Middle ValueRobust
📍 Median — Middle Value

The middle value when data is sorted. Splits the data exactly in half — 50% above, 50% below. Resistant to outliers.

Example · Sorted: 2, 4, 4, 4, 5, 5, 7, 9

n = 8 (even) → average middle two
Middle values: 4 and 5
Median = (4 + 5) / 2 = 4.5
Rules
  • Odd n: middle value at position (n+1)/2
  • Even n: average of the two middle values
  • Always sort first!
  • Preferred for skewed data (e.g. income)

🏆 Mode — Most Frequent Value

The value(s) that appear most often. The only measure of central tendency that works for categorical data.

Types
  • Unimodal: one mode (most common)
  • Bimodal: two modes (two peaks)
  • Multimodal: three or more modes
  • No mode: all values appear once

When to use Mean
  • Data is symmetric, no extreme outliers
  • Calculating further statistics (variance, etc.)
  • Test scores, heights, temperatures
When to use Median
  • Data is skewed or has outliers
  • Income, house prices, wait times
  • Ordinal data (rankings)
Skewness & Central Tendency
  • Symmetric: Mean = Median = Mode
  • Right skew: Mode < Median < Mean
  • Left skew: Mean < Median < Mode
Right skew: long tail → right
e.g. income, reaction times

Left skew: long tail → left
e.g. age at retirement
§2 · Variance & Standard Deviation
Squared Spreadσ²
📐 Variance — Average Squared Deviation

Measures how far each value is from the mean, on average. Squares the deviations so negatives don't cancel out.

Step-by-Step · Dataset: 2, 4, 4, 4, 5, 5, 7, 9 · Mean = 5
Deviations: (2-5)²=9 (4-5)²=1 (4-5)²=1
             (4-5)²=1 (5-5)²=0 (5-5)²=0
             (7-5)²=4 (9-5)²=16
Sum = 9+1+1+1+0+0+4+16 = 32
σ² (pop) = 32 / 8 = 4.0
s² (sample) = 32 / 7 ≈ 4.57
Formulas
σ² = Σ(xᵢ − μ)² / N ← population
s² = Σ(xᵢ − x̄)² / (n−1) ← sample
n−1: Bessel's correction for unbiased estimate
  • Units are squared (e.g. cm²) — hard to interpret
  • Large variance = data is widely spread
Same Unitsσ
📏 Standard Deviation — Root of Variance

The square root of variance. Returns spread to the same units as the data — much easier to interpret than variance.

From above: σ² = 4.0
σ = √4.0 = 2.0

Interpretation: values are on average
±2.0 units away from the mean (5.0)
Formulas
σ = √( Σ(xᵢ−μ)² / N ) ← population
s = √( Σ(xᵢ−x̄)² / (n−1) ) ← sample
Empirical Rule (Normal Distribution)
  • μ ± 1σ contains ≈ 68% of data
  • μ ± 2σ contains ≈ 95% of data
  • μ ± 3σ contains ≈ 99.7% of data
Key DistinctionN vs n−1
👥 Population vs Sample Statistics

Population stats describe the entire group. Sample stats estimate the population from a subset. Use n−1 (Bessel's correction) for unbiased sample estimates.

PopulationSample
Symbol (mean)μx̄
Symbol (std)σs
Symbol (var)σ²s²
DenominatorNn − 1
Use whenFull dataSubset only
# Why n−1?
Sample tends to underestimate spread.
Dividing by (n−1) corrects this bias.
As n→∞, n−1 ≈ n (same result)

  • Simple but ignores all middle values
  • One outlier can distort it completely
RobustQ3 − Q1
📦 IQR — Interquartile Range

Range of the middle 50% of data. Resistant to outliers. Used in box plots to detect outliers.

Sorted: 2, 4, 4 | 4, 5 | 5, 7, 9
Q1 (25th %) = 4
Q3 (75th %) = 6
IQR = Q3 − Q1 = 6 − 4 = 2
Outlier Rule
Lower fence = Q1 − 1.5·IQR
Upper fence = Q3 + 1.5·IQR
Values outside = outliers
Percentiles5-Number Summary
🎯 Quartiles & Percentiles

Quartiles split sorted data into 4 equal parts. The 5-number summary fully describes a distribution's shape.

Q1 = 25th percentile (lower quartile)
Q2 = 50th percentile = Median
Q3 = 75th percentile (upper quartile)

5-Number Summary:
Min, Q1, Median, Q3, Max
  • P90 = 90th percentile → 90% of data below
  • Box plot visualizes the 5-number summary
RelativeScale-free
📊 Coefficient of Variation

Std dev expressed as a % of the mean. Allows comparing spread across datasets with different units or scales.

CV = (σ / μ) × 100%

Example A: μ=100, σ=10 → CV=10%
Example B: μ=5, σ=2 → CV=40%
B is relatively more spread out
  • Useful for comparing risk in finance
  • Only valid when mean > 0
§4 · Z-Score & Python Implementation
StandardizeHow far from mean?
🎯 Z-Score — Standard Score

How many standard deviations a value is from the mean. Standardizes data to a common scale regardless of units.

z = (x − μ) / σ

Dataset μ=5, σ=2 · x=9:
z = (9 − 5) / 2 = +2.0
→ 9 is 2 std devs above the mean

z = 0 → exactly at the mean
z = +1 → 1 std dev above
z = −2 → 2 std devs below
|z| > 2 → potential outlier
  • After z-scoring: μ=0, σ=1 for any dataset
  • Used in normalization before ML models
  • Converts any normal distribution to standard normal
NumPynumpy / scipy
🐍 Python — NumPy & SciPy
import numpy as np
from scipy import stats

data = [2, 4, 4, 4, 5, 5, 7, 9]

# Central Tendency
np.mean(data) # → 5.0
np.median(data) # → 4.5
stats.mode(data).mode # → 4

# Spread
np.var(data) # population var → 4.0
np.var(data, ddof=1) # sample var → 4.57
np.std(data) # population σ → 2.0
np.std(data, ddof=1) # sample s → 2.14
np.ptp(data) # range → 7
np.percentile(data, [25,50,75])

# Z-scores
stats.zscore(data) # all z-scores

§5 · Quick Reference Summary
All Formulas
∑ Formula Sheet
Mean: μ = Σxᵢ / N
Median: middle value (sorted)
Mode: most frequent value
Range: Max − Min
IQR: Q3 − Q1
Var: σ² = Σ(xᵢ−μ)²/N
Std: σ = √σ²
CV: σ/μ × 100%
Z: (x − μ) / σ
Decision Guide
🧭 Which Measure to Use?
  • Symmetric data → Mean + Std Dev
  • Skewed data → Median + IQR
  • Categorical → Mode only
  • Outliers present → Median + IQR
  • Comparing datasets → CV (coefficient of variation)
  • Normalization → Z-score
  • Box plot → 5-number summary
Interpretation
💬 Interpreting Spread
  • Small σ → data clustered near mean
  • Large σ → data widely scattered
  • σ = 0 → all values are identical
  • IQR small → middle 50% is tightly grouped
  • IQR large → high spread in central data
Low spread: 2, 3, 3, 4, 4 σ≈0.8
High spread: 1, 3, 5, 7, 9 σ≈2.8
Outlier Detection
🚨 Detecting Outliers
  • Z-score method: |z| > 2 or 3 = outlier
  • IQR method: outside Q1−1.5·IQR or Q3+1.5·IQR
  • Box plot whiskers show IQR fences
  • Outliers inflate Mean and Std Dev
# IQR outlier detection
lower = Q1 - 1.5*IQR
upper = Q3 + 1.5*IQR
outliers = df[(df<lower)|(df>upper)]
